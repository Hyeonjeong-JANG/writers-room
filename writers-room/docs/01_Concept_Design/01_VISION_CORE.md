# Vision & Core Values: Writer's Room
> Created: 2026-04-25 10:30
> Last Updated: 2026-04-25 10:30

## 1. 핵심 비전

**"독자의 한 줄 댓글이 스토리의 방향을 바꾸고, 그 기여가 투명하게 보상받는 AI 공동 창작 플랫폼"**

Writer's Room은 드라마와 웹소설 창작 과정을 민주화한다. 작가 혼자 고민하던 "다음 회차"를 AI 에이전트들(PD, 작가, 편집자)이 작가방에 모여 토론하고, 독자 커뮤니티의 아이디어가 실시간으로 스토리에 반영된다. 모든 기여와 거래는 Base 블록체인에 기록되어 신뢰할 수 있는 평판 시스템을 형성한다.

## 2. 핵심 가치

| 가치 | 정의 | 실현 방법 |
|:---|:---|:---|
| **투명한 기여 추적** | 모든 참여자의 기여도가 검증 가능하게 기록된다 | Base 온체인 기여 로그 |
| **공정한 보상** | 아이디어의 가치는 사용된 만큼 보상받는다 | x402 USDC 결제, 기여도 기반 분배 |
| **창작의 민주화** | 누구나 스토리에 영향을 미칠 수 있다 | 댓글 기반 아이디어 선별 시스템 |
| **AI 협업** | AI는 도구가 아닌 창작 파트너다 | 역할별 전문 에이전트 (PD, 작가, 편집자) |
| **오픈 에이전트 경제** | 유저가 만든 에이전트가 경제적 가치를 가진다 | 에이전트 마켓플레이스, 고용/평판 시스템 |

## 3. 타겟 오디언스

### Primary: 창작자 (Story Creators)

- **프로필**: 드라마/웹소설 작가 지망생, 인디 창작자, 웹소설 플랫폼 활동 작가
- **Pain Point**: 혼자 쓰는 창작의 고립감, 독자 피드백 반영의 어려움, 다음 회차 아이디어 고갈
- **현재 해결 방식**: 작가 커뮤니티 게시판, ChatGPT에 아이디어 요청, 독자 댓글 수동 분석
- **기대 가치**: AI 작가방이 브레인스토밍 파트너 역할, 독자 의견 자동 선별 및 반영

### Secondary: 독자/커뮤니티 (Readers & Contributors)

- **프로필**: 웹소설/웹툰 구독자, 댓글 활발 참여자, 2차 창작 활동가
- **Pain Point**: 댓글을 달아도 스토리에 반영되지 않음, 기여에 대한 인정 부재
- **현재 해결 방식**: 팬픽션, 댓글에 전개 예측 작성, SNS에서 작가에게 DM
- **기대 가치**: 내 아이디어가 실제 스토리에 반영되고 온체인에 기록됨

### Tertiary: 에이전트 빌더 (Agent Builders)

- **프로필**: AI/프롬프트 엔지니어, 장르 전문가, 크리에이터 이코노미 참여자
- **Pain Point**: 만든 AI 에이전트를 수익화할 방법이 없음
- **현재 해결 방식**: GPTs Store (수익 미미), 프롬프트 마켓플레이스 (일회성)
- **기대 가치**: 에이전트를 훈련시켜 고용당할 때마다 x402 USDC 수익

## 4. 성공의 정의

### 단기 (3개월)
- MVP 출시, 초기 스토리 10편 이상 운영
- AI 에이전트 작가방 토론 -> 챕터 생성 파이프라인 안정화
- 첫 x402 USDC 거래 발생

### 중기 (6개월)
- 월간 활성 스토리 100편
- 에이전트 마켓플레이스 오픈, 커스텀 에이전트 50개 이상 등록
- 온체인 기여 기록 1,000건 이상

### 장기 (12개월)
- 에이전트 고용 거래량 월 $10K+ USDC
- 커뮤니티 기여로 만들어진 스토리의 영상화/출판 첫 사례
- Base 생태계 내 크리에이터 이코노미 대표 프로젝트

## 5. 비전 검증 체크리스트

- [ ] 독자 댓글이 실제로 스토리 방향에 영향을 미치는가?
- [ ] AI 에이전트 토론이 사람이 읽기에 자연스럽고 유용한가?
- [ ] x402 결제가 매끄럽게 작동하는가? (3클릭 이내)
- [ ] 온체인 기여 기록이 투명하게 조회 가능한가?
- [ ] 에이전트 빌더가 실제로 수익을 얻고 있는가?
