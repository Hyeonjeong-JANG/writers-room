## 3. Unique Value Proposition (UVP)

**"당신의 댓글이 다음 회차를 바꿉니다 -- AI 작가방에서 함께 만드는 스토리, 기여는 온체인에 영원히"**

- 창작자에게: AI 에이전트 팀(PD, 작가, 편집자)이 당신의 작가방에서 24/7 브레인스토밍
- 독자에게: 댓글 하나로 스토리의 방향을 바꾸고, 기여도가 영구 기록
- 에이전트 빌더에게: 만든 에이전트가 고용될 때마다 USDC 수익

## 4. Solution

| 문제 | 솔루션 | 핵심 기술 |
|:---|:---|:---|
| P1 아이디어 고갈 | AI 작가방: PD/작가/편집자 에이전트가 맥락을 유지하며 토론하고 다음 챕터 초안 생성. Selanet이 실시간 장르 트렌드를 수집해 PD 에이전트에 주입 | FLock API (LLM 호출), Selanet (웹 트렌드 수집), 멀티 에이전트 오케스트레이션 |
| P2 독자 기여 미반영 | 커뮤니티 댓글에서 흥미로운 아이디어를 AI가 선별, 작가방 토론에 투입, 채택 시 기여도 온체인 기록 | NLP 댓글 분석, Base 온체인 기여 로그 |
| P3 에이전트 수익화 부재 | 에이전트 마켓플레이스: 장르별 커스텀 에이전트 생성/훈련, 타인이 고용 시 x402 USDC 자동 결제. Nansen Trust Score로 에이전트 신뢰도 검증 | x402 프로토콜, Nansen API (온체인 행동 분석), Coinbase Smart Wallet |

## 6. Revenue Streams (수익 모델)

| 수익원 | 설명 | 예상 단가 |
|:---|:---|:---|
| **에이전트 고용 수수료** | 타인의 에이전트를 고용할 때 x402 USDC 결제. 플랫폼이 수수료(예: 10%) 차감 후 빌더에게 지급 | $0.5 ~ $5 / 고용 건 |
| **프리미엄 에이전트 슬롯** | 기본 3개 에이전트 무료, 추가 슬롯 또는 고급 에이전트(더 큰 모델) 유료 | $3 ~ $10 / 월 |
| **스토리 발행 수수료** | 완성된 스토리를 NFT로 발행하거나 출판할 때 플랫폼 수수료 | 발행 수익의 5% |

## 8. Key Metrics

| 지표 | 정의 | 목표 (3개월) |
|:---|:---|:---|
| **활성 스토리 수** | 주 1회 이상 새 챕터가 생성되는 스토리 | 50 |
| **댓글 채택률** | 전체 댓글 중 에이전트 토론에 투입된 비율 | 5% |
| **에이전트 고용 건수** | x402로 결제된 에이전트 고용 수 | 200건/월 |
| **온체인 기여 기록 수** | Base에 기록된 기여 트랜잭션 수 | 500건 |
| **에이전트 빌더 수** | 커스텀 에이전트를 등록한 유저 수 | 30명 |

## 9. Unfair Advantage (경쟁 해자)

1. **온체인 기여 데이터 축적**: 시간이 지날수록 기여 기록이 쌓여 에이전트 평판과 작품 히스토리가 복제 불가능한 자산이 된다
2. **네트워크 효과**: 좋은 에이전트 -> 더 많은 고용 -> 빌더 수익 증가 -> 더 좋은 에이전트 유입 -> 더 많은 작가 유입
3. **x402 + Base 선점**: 크리에이터 이코노미에서 x402 결제를 가장 먼저 적용한 사례로 Coinbase/Base 생태계 내 포지셔닝
