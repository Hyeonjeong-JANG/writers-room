## 2. 화면별 상세 플로우

### Flow A: 온보딩 (첫 방문 -> 첫 챕터 생성)

목표: 가입 후 3분 내 첫 챕터 생성 경험

```
Step 1: 랜딩 페이지
  - Hero: "AI 작가방에서 함께 만드는 스토리"
  - CTA: "지금 시작하기"
  - 하단: 작동 방식 3단계 설명, 샘플 스토리 미리보기

Step 2: Smart Wallet 연결
  - Coinbase Smart Wallet 팝업
  - 소셜 로그인 (Google, Apple) 또는 패스키
  - 지갑 자동 생성 (가스리스)
  - 연결 완료 -> 자동으로 Step 3으로 이동

Step 3: 첫 스토리 생성 가이드
  - 장르 선택 카드 (로맨스, 판타지, 스릴러, SF, 드라마)
  - 제목 입력
  - 시놉시스 입력 (또는 AI가 장르 기반 추천)
  - 핵심 캐릭터 2-3명 설정 (이름, 성격, 관계)
  - "작가방 열기" CTA

Step 4: 작가방 첫 토론
  - 기본 에이전트 3명 자동 배치 (PD, 작가, 편집자)
  - "첫 챕터 토론 시작" 버튼
  - 토론 진행 중 로딩 애니메이션 (에이전트 아바타 + 진행률)
  - 토론 요약 -> 챕터 초안 확인 -> 발행
```

### Flow B: 챕터 읽기 -> 댓글 기여

```
Step 1: 스토리 탐색
  - 카드 그리드: 썸네일, 제목, 장르 태그, 최신 챕터 번호
  - 필터: 장르, 연재 상태, 인기순/최신순
  - 검색

Step 2: 스토리 상세
  - 시놉시스, 캐릭터 소개
  - 챕터 목록 (번호, 제목, 작성일)
  - 작가방 참여 에이전트 목록
  - 커뮤니티 통계 (댓글 수, 기여자 수)

Step 3: 챕터 읽기
  - 텍스트 중심 레이아웃 (웹소설 리더 스타일)
  - 하단 고정: 이전/다음 챕터 네비게이션
  - 읽기 완료 후 댓글 섹션으로 스크롤

Step 4: 댓글 섹션
  - 일반 댓글 탭 / 아이디어 제안 탭 분리
  - 아이디어 제안 시 태그 선택: #전개제안 #캐릭터제안 #설정제안
  - 좋아요/반응 기능
  - "AI 선별됨" 배지 (작가방 토론에 투입된 댓글)
  - "이 아이디어가 챕터 N에 반영되었습니다" 알림 배너
```

### Flow C: 에이전트 마켓플레이스

```
Step 1: 마켓플레이스 탐색
  - 필터: 역할(PD/작가/편집자), 장르, 평점, 가격대
  - 에이전트 카드: 아바타, 이름, 역할, 장르 태그, 평점, 고용 횟수, 가격

Step 2: 에이전트 상세
  - 프로필: 아바타, 이름, 역할, 전문 장르
  - 평점 (1-5) + 리뷰 목록
  - 참여 작품 목록 (링크)
  - 고용 이력 통계 (총 고용 수, 최근 30일)
  - 샘플 출력 미리보기
  - "고용하기" CTA + 가격 표시 (USDC)

Step 3: 고용 결제
  - x402 USDC 결제 확인 팝업
  - Smart Wallet 서명
  - 결제 완료 -> 에이전트가 내 작가방에 추가됨
  - 온체인 영수증 링크

Step 4: 에이전트 생성
  - 역할 선택 (PD / 작가 / 편집자)
  - 장르 전문성 선택 (복수 가능)
  - 아바타 선택 또는 업로드
  - 이름, 소개 작성
  - 시스템 프롬프트 작성 (가이드라인 제공)
  - 샘플 작품 업로드 (few-shot examples)
  - 가격 설정 (USDC)
  - 미리보기 테스트 -> 등록
```

### Flow D: 대시보드

```
대시보드 메인
  |
  +-- 내 스토리
  |     - 스토리 카드 목록 (상태: 연재중/휴재/완결)
  |     - 각 스토리별: 챕터 수, 조회수, 댓글 수
  |     - "작가방 열기" 바로가기
  |
  +-- 내 에이전트
  |     - 등록한 에이전트 목록
  |     - 각 에이전트별: 평점, 고용 횟수, 총 수익
  |     - "수정" / "비활성화" 관리
  |
  +-- 수익 현황
  |     - 총 수익 (USDC)
  |     - 에이전트별 수익 차트
  |     - 최근 거래 내역 (x402 트랜잭션 목록)
  |     - Base Explorer 링크
  |
  +-- 기여 히스토리
        - 내 댓글 중 채택된 목록
        - 기여한 스토리/챕터 링크
        - 온체인 기여 기록 링크
```

## 3. 모바일 반응형 고려사항

| 화면 | 데스크톱 | 모바일 |
|:---|:---|:---|
| GNB | 가로 메뉴 (스토리, 에이전트, 대시보드, 알림, 지갑) | 하단 탭 바 (5개 아이콘) |
| 작가방 | 좌측 에이전트 목록 + 우측 채팅 영역 | 전체화면 채팅, 에이전트 목록은 드로어 |
| 스토리 탐색 | 3-4열 카드 그리드 | 1열 카드 리스트 |
| 챕터 읽기 | 좌우 여백 넓은 텍스트 | 풀스크린 리더 모드 |
| 에이전트 상세 | 좌측 프로필 + 우측 리뷰/이력 | 탭 전환 (프로필 / 리뷰 / 이력) |
