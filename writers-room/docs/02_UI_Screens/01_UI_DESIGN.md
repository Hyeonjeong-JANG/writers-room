> Created: 2026-04-25 10:45

## 1. 디자인 원칙

| 원칙 | 설명 |
|:---|:---|
| **콘텐츠 우선** | 스토리 텍스트가 주인공. UI는 콘텐츠를 방해하지 않는다 |
| **친근하고 밝은** | 라이트 모드 중심. 웹소설 플랫폼의 친근함 유지 |
| **점진적 복잡성** | 첫 화면은 단순, 깊이 들어갈수록 고급 기능 노출 |
| **온체인 투명성** | 블록체인 기록은 자연스럽게 노출하되 강제하지 않음 |

## 5. 핵심 컴포넌트

### 5.1 Global Navigation Bar (GNB)

```
+-----------------------------------------------------------------------+
| [Logo]   스토리   에이전트   대시보드          [알림 bell]  [지갑 USDC] |
+-----------------------------------------------------------------------+
```

- 위치: 상단 고정 (sticky)
- 높이: 64px
- 배경: `#FFFFFF` + 하단 border `#E5E7EB`
- 로고: 좌측
- 메뉴: 중앙 (스토리, 에이전트, 대시보드)
- 우측: 알림 벨 아이콘 (뱃지), 지갑 버튼 (USDC 잔액 표시)
- 모바일: 하단 탭 바로 전환

### 5.2 스토리 카드

```
+---------------------------+
| [썸네일 이미지]            |
|                           |
| 로맨스  판타지             |  <- 장르 태그
| 달빛 아래의 약속           |  <- 제목 (H3)
| 12화 연재중               |  <- 상태
| ★ 4.2  |  👁 1.2K  |  💬 89 |  <- 메타
+---------------------------+
```

- 크기: 데스크톱 280px 너비, 모바일 100% 너비
- 썸네일: 16:9 비율, border-radius 12px
- 장르 태그: 작은 pill 형태, Primary Light 배경
- 호버: shadow-md + 약간 위로 이동 (translateY -2px)

### 5.3 작가방 (채팅 스타일)

```
+-------------------+----------------------------------------+
| 에이전트 목록      | 채팅 영역                                |
|                   |                                        |
| [PD] 김PD         | [PD 김PD]                    10:23     |
|   ★ 4.5           | 이번 회차는 주인공의 내적 갈등을        |
|                   | 중심으로 전개하는 게 좋겠습니다.         |
| [작가] 이작가      |                                        |
|   ★ 4.8           | [작가 이작가]                  10:24     |
|                   | 동의합니다. 독자 댓글 중에 "비밀이      |
| [편집자] 박편집    | 드러나는 장면" 제안이 있었는데...        |
|   ★ 4.2           |                                        |
|                   | [편집자 박편집]                 10:24     |
| ──────────        | 타이밍은 좋지만 복선을 먼저             |
| 채택된 댓글        | 깔아야 합니다. 10화에서 힌트를...        |
|                   |                                        |
| "비밀이 드러나는   | ──────── 토론 요약 ────────            |
|  장면이 있으면     | 핵심 결정: ...                         |
|  좋겠어요" - @user | 다음 챕터 방향: ...                     |
|                   |                                        |
|                   | [챕터 초안 생성하기]                     |
+-------------------+----------------------------------------+
```

- 좌측 사이드바 (280px): 에이전트 목록 + 채택된 댓글
- 우측 메인 영역: 채팅 버블 형태의 토론 로그
- 에이전트별 버블 색상 구분 (역할별 Semantic Color)
- 토론 요약 섹션: 구분선 아래 하이라이트 박스
- 하단: "챕터 초안 생성하기" CTA 버튼
- 모바일: 전체화면 채팅, 사이드바는 좌측 드로어

### 5.4 챕터 리더

```
+--------------------------------------------------+
|                                                  |
|          [스토리 제목] - 12화                      |
|                                                  |
|     스토리 본문 텍스트가 여기에 표시됩니다.          |
|     Noto Serif KR 18px, 행간 1.8                  |
|     최대 너비 680px, 중앙 정렬                     |
|                                                  |
|     ...                                          |
|                                                  |
+--------------------------------------------------+
| [< 이전화]            12/15            [다음화 >] |
+--------------------------------------------------+
| 댓글 섹션                                         |
| [일반 댓글] [아이디어 제안]                         |
+--------------------------------------------------+
```

- 본문 영역: 최대 너비 680px, 중앙 정렬
- 폰트: Noto Serif KR 18px, 행간 1.8 (독서 최적화)
- 하단 네비게이션: sticky, 이전/다음 챕터
- 댓글 섹션: 2개 탭 (일반 댓글 / 아이디어 제안)

### 5.5 에이전트 프로필 카드

```
+---------------------------+
| [아바타]                   |
| 이작가                     |  <- 이름
| ★ 4.8 (127 리뷰)          |  <- 평점
| 작가 | 로맨스, 판타지       |  <- 역할 + 장르
| 342회 고용                 |
|                           |
| $1.50 USDC / 고용          |  <- 가격
| [고용하기]                 |  <- CTA
+---------------------------+
```

```
|  총 결제: 1.50 USDC                   |

|                                       |

```

## 6. 인터랙션 패턴

| 패턴 | 사용처 | 동작 |
|:---|:---|:---|
| **토론 진행 중** | 작가방 | 에이전트 아바타에 타이핑 애니메이션, 프로그레스 바 |
| **댓글 채택** | 챕터 댓글 | Success 컬러 뱃지 + 토스트 알림 |
| **결제 완료** | x402 결제 | 체크마크 애니메이션 + 온체인 영수증 링크 |
| **에이전트 추가** | 작가방 | 에이전트가 채팅방에 "입장" 메시지와 함께 추가 |
| **챕터 발행** | 작가방 | 컨페티 애니메이션 + "N화가 발행되었습니다" 토스트 |
| **알림** | 전역 | 벨 아이콘 뱃지 숫자, 드롭다운 알림 패널 |

## 7. 반응형 브레이크포인트

| 이름 | 범위 | 레이아웃 변화 |
|:---|:---|:---|
| Mobile | 0 - 639px | 1열, 하단 탭 바, 풀스크린 채팅 |
| Tablet | 640 - 1023px | 2열 그리드, 축소된 사이드바 |
| Desktop | 1024px+ | 3-4열 그리드, 풀 사이드바 |

Tailwind 기준: `sm:640px`, `md:768px`, `lg:1024px`, `xl:1280px`

## 8. 접근성 가이드라인

- 색상 대비: WCAG AA 기준 (4.5:1) 충족
- 포커스 표시: `ring-2 ring-indigo-500 ring-offset-2`
- 키보드 네비게이션: 모든 인터랙티브 요소 Tab 접근 가능
- 스크린 리더: 에이전트 채팅 버블에 role="log", aria-live="polite"
- 모션 감소: `prefers-reduced-motion` 미디어 쿼리 지원
